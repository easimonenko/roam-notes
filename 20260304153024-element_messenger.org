*Element Messenger* -- [[id:acc2a94c-32ea-40c4-86a0-d8de3085f574][свободно-распространяемое]] федеративное приложение для обмена сообщениями на основе протокола [[id:33296091-e9f5-4f52-8dba-33c18fea3bca][Matrix]]. Выпускаются две варианта: для настольного компьтера и смартфона. Вариант для ПК написан на [[id:ef72bec4-29ee-43e3-a8f1-6ac4594233e8][JavaScript]] с использованием Electron. Вариант для смартфона в зависимости от операционной системы: Java, Kotlin, Objective-C, Swift. Распространяется под [[id:9541deca-d668-45d6-9a8e-c295d2435c2f][GNU GPL License]].
