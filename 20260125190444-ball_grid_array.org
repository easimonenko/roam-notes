*Ball Grid Array* (BGA) -- тип корпусов [[id:d5f7a0e3-5ca2-4873-b1c3-7906bb133a11][микросхем]], предназначенных для припаивания к [[id:79fc3bb2-5473-4295-be63-a0d26c0b93dd][плате]]. Контакты образуют матрицу расплавляемых шариков. Широко используются в современных микросхемах с большим количеством выводов.
