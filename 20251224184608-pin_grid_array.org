*Pin Grid Array* (PGA) -- разновидность корпусов для [[id:e7cbfa8e-528f-4ae2-b508-b5d717e7ecb6][интегральных схем]] со штырьковыми массивами выводов по всей или части поверхности корпуса чипа. Широко применялись для [[id:cf8e77c1-1b45-44ad-9682-8f2fc7c52792][микропроцессоров]].
